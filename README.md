# Guachinche El Realejo — estructura del repo

Este repositorio está organizado por capas para mantener el proyecto más claro.

## Carpetas

- `frontend/`: aplicación React + Vite (UI)
- `backend/`: backend (pendiente / placeholder)
- `database/`: base de datos / seeds / documentación (placeholder)

## Comandos (desde la raíz)

```bash
npm run dev
npm run build
npm run preview
```

> Los scripts de la raíz delegan en `frontend/`.
